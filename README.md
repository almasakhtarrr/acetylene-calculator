Acetylene Gas Calculator is a web-based application built using Streamlit that calculates the volume of acetylene gas produced from calcium carbide using fundamental chemical principles and the Ideal Gas Law (PV=nRT).

The app allows users to input the mass of calcium carbide along with temperature and pressure conditions to compute the theoretical volume of acetylene gas. It also provides a graphical visualization of how gas volume varies with temperature, making it a useful educational and analytical tool for students and engineering learners.

The application emphasizes safe, theoretical understanding and includes a built-in safety information panel to promote responsible scientific practices.

✨ Key Features
⚗️ Based on real chemical reaction (CaC₂ + H₂O → C₂H₂)
📐 Uses Ideal Gas Law for accurate calculations
📊 Dynamic graph: Volume vs Temperature
🌡️ Custom temperature & pressure inputs
⚠️ Safety information panel
☁️ Deployed using Streamlit Cloud
💻 Open-source and easy to use
